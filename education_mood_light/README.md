# 아두이노 무드등 제작 교육
아두이노를 활용하여 무드등 제작 교육

아두이노와 RGBLED를 활용하여 무드등 제작   
터치센서와 부저를 활용해 ON/OFF 제어 및 표시

## 하드웨어  
아두이노(uno)  
부저  
RGBLED  
터치센서(TTP223)  

## 소프트웨어  
아두이노 IDE
