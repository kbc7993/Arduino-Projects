# 아두이노 교육용 키트 제작 및 교육
IoT란 주제에 맞는 교육용 키트를 아두이노로 제작

자동에어컨제어, 가스경보기, 원격전등제어기, 자동쓰레기통, 방범시스템

## 하드웨어  
아두이노(nano)
led  
서보모터(SG90)  
초음파센서(HC-SR04)  
부저  
가스감지센서(MQ-2)  
블루투스모듈(HC-06)  
온습도센서(DHT11)

## 소프트웨어  
아두이노 IDE
